Wie erstelle ich ein Content Element?

  • Um ein Content Element anzulegen benötigst du nur 2 Dateien.
  • 1. Erstelle eine Config-Datei unter /typo3conf/ext/dw_content_elements_source/Configuration/Elements/E1000MYContentElement.ts
  • In der Config-Datei werden alle Einstellungen vorgenommen z.b. der Titel und Felder gesetzt.
  • Die Config-Datei könnte so aussehen:
    (Durch Screenshot mit Config und Dateibaum ersetzen) title = DWCE List
    description = List Description
    icon = ../../typo3conf/ext/dw_content_elements_source/Resources/Public/Thumbnails/E1000List.png
    fields = --palette--;Headline;headerText, tx_dwc_list_item
    overWriteShowitem = 0
  • Erklärung der Einstellungs möglichkeiten
  • 2. Das Fluid Template diese Datei muss genau so wie die Config-Datei heißen und liegt unter /typo3conf/ext/dw_content_elements_source/Resources/Private/Templates/Elements/E1000MYContentElement.html
  • Hier kannst du das HTML und Fluid einfügen und die Frontend Ausgabe bauen, alle Datten des Content Elements kannst du über die Variable "data" auslesen.
  • So könnte das Fluid Template aussehen:
    (Screenshot des Fluid Templates und Dateibaum)
  • Jetzt musst du noch einmal den Cache Leeren und dann kannst du das Content Element benutzen ;)

Wie erstelle ich ein IRRE Content Element?

  • Um ein IRRE Content Element zu erstellen legen Sie zuerst ein normales an.
  • Jetzt legen Sie in der /typo3conf/ext/dw_content_elements_source/ext_tables.php die IRRE Tabelle und das entsprechende TCA Feld für die tt_content Tabelle an.
  • Nun fügen Sie die Datenbankfeld in der /typo3conf/ext/dw_content_elements_source/ext_tables.sql hinzu.
  • Dann Konfigurieren Sie das TCA für die IRRE Tabelle und speichenr dieses unter /typo3conf/ext/dw_content_elements_source/Configuration/TCA/IRRE ab.
  • Als letztes legen Sie in der /typo3conf/ext/dw_content_elements_source/Classes/Controller/ElementsController.php noch eine Action mit dem Namen ihres Content Elements an wo soe folgende Zeile einfügen:
  • $this->view->assign('irreRelations', $this->irreService->getRelations($this->contentObj, 'NAMEDERIRRETABELLE'));
  • Dadurch haben Sie nun in ihren Fluid Template eine Array Variable names "irreRelations" in dem Ihre IRRE gespeichert ist.

Wie setze/nutze ich Settings?

  • Um Settings oder globale Constanten nutzen zu können,
    musst du diese nur in die /typo3conf/ext/dw_content_elements_source/Configuration/TypoScript/setup.txt eintragen.
  • Dadurch kannst du diese in Fluid Template über die Variable "settings" abrufen.
